# Name Check — "Orrery" (Phase 2 deliverable)

**Verdict: usable, low legal risk, moderate crowding.** Formal USPTO/EUIPO search still required before Phase 6 public launch (R10).

## Findings (2026-07-12 web search)
- "Orrery" is a **generic English word** (the 18th-century mechanical solar-system model) — genericness cuts both ways: hard for anyone to monopolize, hard for us to trademark broadly (a stylized mark/logo is the path).
- Existing uses found — all small, none a dominant commercial product:
  - NASA's Eyes "Solar System Interactive" is informally called *the Orrery* and lives at `eyes.nasa.gov/apps/orrery` — the most notable neighbor (confusion risk, not legal risk; NASA doesn't trademark-enforce hobby words).
  - Open-source GNU/Linux phone astronomy app "orrery" (GitHub, niche).
  - Two dated Android apps ("Orrery" by trideon.de 2013; "Copernican Orrery"), an itch.io game, a hobbyist "Orrery Software" brand, a Rust architecture-diagram DSL.
- No evidence found of a live software trademark blocking the space; not a legal opinion.

## Implications
- **SEO:** "orrery" queries today mean the antique device — we'd own the term's software meaning over time (like "Notion", "Figma" — generic words won). Brand searches will need "orrery app / orrery space" until established.
- **Domains:** `orrery.com` almost certainly taken/parked. Realistic: `orrery.space` (on-brand TLD), `orrery.app`, `useorrery.com`, `orrery.sky`-style. Decision at Phase 5 (domain purchase).
- **Alternatives** if the owner prefers distinctiveness (D1): **Skygraph** (says the thesis), **Astrary**, **Everysky**. Recommendation: **keep Orrery** — evocative, memorable, meaning-aligned; revisit only if domain/TM search fails.
